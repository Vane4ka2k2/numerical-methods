# numerical-methods

Консольное C++ приложение для численного поиска корней нелинейных уравнений с использованием алгебраических многочленов (полиномов). Проект демонстрирует реализацию классических численных методов и объектно-ориентированную модель многочлена.

## 🧮 Реализованные численные методы
1. **Метод Ньютона (метод касательных)** (`methodNewton`): Итерационный метод поиска корней с использованием первой и второй производной для определения начальной точки сходимости.
2. **Метод хорд и касательных** (`methodChordsAndTangents`): Комбинированный метод, зажимающий корень с двух сторон (метод хорд с одной стороны и метод касательных с другой) для ускоренной сходимости.

## 📦 Возможности класса `Polynomial`
Класс `Polynomial` инкапсулирует математические операции над многочленами произвольной степени:
- **Арифметические операции**: Сложение (`+`), вычитание (`-`), умножение (`*`), деление нацело (`/`) и вычисление остатка от деления (`%`) для многочленов и вещественных чисел (скаляров).
- **Сравнения**: Логические операторы равенства (`==`, `!=`) и отношения порядка (`>`).
- **Специальные методы**:
  - `derivative()` — вычисление аналитической производной многочлена.
  - `gcd(other)` — нахождение наибольшего общего делителя (НОД) двух многочленов.
  - `removeMultiplicity()` — устранение кратности корней полинома.
  - `roundCoefficients()` — округление числовых коэффициентов до заданной точности (3 знака после запятой).
- **Потоковый ввод-вывод**: Перегрузка операторов `<<` и `>>` для форматированного вывода вида `3x^3 - 24x^2 + 1` в консоль.

## 🚀 Сборка и запуск

Для компиляции проекта вам потребуется компилятор с поддержкой стандарта C++11 или выше (например, GCC или MSVC).

### Пример компиляции через GCC (g++):
```bash
g++ -std=c++11 main.cpp -o num_methods
```

### Пример запуска:
```bash
./num_methods
```

Программа выполнит тестовый расчет корней полинома $p(x) = x^3 + 3x^2 - 24x + 1$ на заданных интервалах и выведет результаты в консоль:
```text
p2: 1x^3 + 3x^2 - 24x + 1
Корень: -7.042
Корень: 3.829
```
