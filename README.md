## 🌟 Реализованные методы

- 🎯 **Метод Ньютона (касательных)** — Поиск корней с высокой скоростью сходимости на основе производных.
- 📏 **Метод хорд (секущих)** — Итерационный метод уточнения корней на отрезке.
- 🧮 **Полиномиальная алгебра** — Вычисление значений многочленов и их производных.
